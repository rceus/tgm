# Terminal Gmail - A Gmail CLI

A command line tool for Gmail. (In production)

## Installation

Add this line to your application's Gemfile:

    gem 'tgm'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tgm

** Uninstall the gmail gem and install it from the repository directly. (manually) 

## Usage

Current Commands:

	tgm help [COMMAND]

	tgm launch

	tgm login

	tgm labels

	tgm mail

	tgm from
